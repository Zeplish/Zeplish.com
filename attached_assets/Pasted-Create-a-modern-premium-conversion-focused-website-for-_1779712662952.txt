Create a modern, premium, conversion-focused website for “Zeplish” — a custom AI-powered business software development company for SMEs.

Core positioning:
Zeplish builds custom AI-powered business software for small and medium businesses that are tired of using spreadsheets, WhatsApp follow-ups, manual tracking, and disconnected tools.

Main headline:
Custom AI Software for SMEs That Want to Save Time, Reduce Manual Work & Grow Faster

Subheadline:
We design and build practical AI-powered business applications, dashboards, CRMs, automation systems, and internal tools tailored to your exact workflow — without the complexity of enterprise software.

Website goal:
Generate leads from SME owners, service businesses, schools, clinics, agencies, consultants, manufacturers, property managers, and local businesses who need custom software but don’t know where to start.

Design style:
- Premium, clean, trustworthy, modern SaaS agency website
- Use a white/light background with dark text and one strong accent color
- Rounded cards, soft shadows, clean icons, smooth sections
- Mobile responsive
- Simple animations on scroll
- Clear CTAs throughout
- Use professional but simple language, not too technical

Tech stack:
Use React with Tailwind CSS. Create a polished single-page website with smooth navigation.
